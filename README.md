# LawGX AI

Premium legal-tech website for LawGX built with Next.js, TypeScript, Tailwind CSS, App Router, and a secure OpenAI Responses API route.

## Local setup

1. Install dependencies:

```bash
npm install
```

2. Create `.env.local` and add:

```bash
OPENAI_API_KEY=your_openai_api_key_here
```

3. Run locally:

```bash
npm run dev
```

4. Build for production:

```bash
npm run build
```
